## Overview

카카오톡 플러스친구 API를 활용해 친구와의 플레이리스트 공유 서비스

![remocon](./readme_assets/remocon.png)

## Usage

`카카오톡`에서 `뮤직키친` 검색 후 친구추가하여 이용 가능

## Feature

* 전광판을 위해 만들어진 서비스로 글자 크기가 크다
* 비동기처리로 새로고침하지 않고 사용
* 향후 플레이리스트 참가자들의 노래목록을 통계로 확인 가능
* 관리자를 위한 노래 조작 리모콘 페이지 제공
